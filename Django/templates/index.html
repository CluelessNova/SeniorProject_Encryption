Encrypt Decrypt Thesis

Jacob Levinky's Senior Project

What is my project about?

My Senior Project is about encryption and this website is for showing off my work. My encryption works by implementing the ceasar cipher to encrypt the message. It takes in a key to shift each letter of the alphabet and mix up the message.

The ceasar cipher takes the alphabet and shifts it a certain amount. The key has a range between 1 and 25, the amount the letters get shifts, so if it the letter is 'A' and the shift key is 4, the new letter from the ceasar cipher is 'E'. The length of the key is the secureness level, and the enryption will cycle through the key and shift the message according.

How to use the website

The encrypt page

  1. Generate a key
    • Use the slider or input a number to change the secureness of the key
    • The key secureness is the length of the key (how many numbers are in the key)
  2. Write your message in the top box
  3. Input the key
    • The key must look equivalent as it was generated
    • It must have spaces inbetween each other number
  4. The encrypted message will be printed at the bottom

The decrypt page

  1. Write the encrypted message in the top box
  2. Input the same key that was used to encrypt the original message
    • The key must look equivalent as it was generated
    • It must have spaces inbetween each other number
  3. The decrypted message will be printed at the bottom
